# Jogo-da-Velha-CP2-Python-
Implementação de um jogo da velha (tic-tac-toe) em Python usando uma matriz 3x3 para representar o tabuleiro. O jogo deve ser para dois jogadores, onde cada jogador pode escolher entre X e O. O tabuleiro deve ser exibido após cada jogada e o jogo deve verificar se houve um vencedor ou se houve um empate.
